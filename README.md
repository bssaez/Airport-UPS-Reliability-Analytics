# Airport UPS Reliability Analytics

Business Intelligence project developed using historical UPS event logs from a real airport maintenance environment.

---

## 📌 Overview

This project automates the integration and analysis of historical event logs generated by multiple Eaton UPS systems.

The solution consolidates data exported from individual UPS units, transforms it using **Power Query**, models it using a **Star Schema**, and presents interactive dashboards in **Power BI** to support maintenance analysis and decision-making.

---

## 🎯 Business Problem

During preventive maintenance activities, each UPS exports its historical event log as an independent file.

Analyzing these files manually is time-consuming and makes it difficult to:

- Identify recurring alarms.
- Compare electrical rooms.
- Analyze historical trends.
- Support maintenance planning.

A centralized analytical solution was required to transform raw event logs into meaningful operational information.

---

## 💡 Solution

A complete Business Intelligence solution was developed including:

- Automated ETL process using Power Query.
- Data cleansing and standardization.
- Star Schema dimensional model.
- DAX measures for KPI calculation.
- Executive dashboard for management.
- Technical dashboard for maintenance personnel.
- Drill-through navigation for detailed event analysis.

## ⚙️ ETL Process

The ETL process performs the following tasks automatically:

- Imports all UPS log files from a folder.
- Removes unnecessary rows.
- Promotes headers.
- Splits Code and Type fields.
- Standardizes event descriptions.
- Converts date formats.
- Creates a consolidated fact table.
- Adds UPS (Electrical Room) identification.

---

## ⭐ Data Model

Star Schema composed of:

### Fact Table

- FactEventos

### Dimensions

- DimCalendario
- DimUPS

Relationships:

- DimCalendario (1) → FactEventos (*)
- DimUPS (1) → FactEventos (*)
